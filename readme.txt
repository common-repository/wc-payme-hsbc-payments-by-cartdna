= Integrate Payme Payments by HSBC with WooCommerce Wordpress store =  

This Plugin adds Payme Payments by HSBC as a payment gateway for WooCommerce.
You must have an active Payme account with enabled Payme payments to use our plugin.
It uses the redirect method. SSL is not mandatory.

= Main features =

* Go cashless. Share bills with friends in just a few taps, no matter which Hong Kong bank they’re with. All you need is a mobile number – no fees, no hassles
